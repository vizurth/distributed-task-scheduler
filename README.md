# Distributed Task Queue & Job Scheduler

gRPC микросервис для распределённого выполнения асинхронных задач. Система управляет очередью, автоматически распределяет задачи между workers и отслеживает статус выполнения в реальном времени.

## Описание

Высокопроизводительная система обработки фоновых задач с поддержкой приоритизации, retry логики и graceful shutdown.

## Технологии

- Go 1.21+
- gRPC & Protocol Buffers
- Apache Kafka
- PostgreSQL
- Redis
- Prometheus & Grafana
- Docker & Docker Compose
